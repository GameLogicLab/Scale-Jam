# Giant vs Tiny — Scale Jam 2026

A third-person puzzle-platformer built for **Kenney Jam 2026**, themed around **SCALE**.

Grow into a giant to push heavy obstacles out of your way, or shrink down to slip through gaps too narrow for anyone else. Solve each level by switching between sizes at just the right moment.

## Gameplay

- **WASD** — Move
- **Mouse** — Look around
- **Space** — Jump
- **Left Shift (hold)** — Grow
- **Left Ctrl (hold)** — Shrink
- **Esc** — Toggle cursor lock

### Core Mechanic
- **Tiny** — faster movement, weaker jump, fits through narrow gaps
- **Giant** — slower movement, stronger jump, can push heavy blocks out of the way

### Objective
Each level has one or more keys scattered around. Collect all the keys and reach the door to complete the level.

## Levels

| Level | Description |
|---|---|
| Level 1 | Tutorial — introduces the grow/shrink mechanic |
| Level 2 | Combines size-switching with key collection puzzles |
